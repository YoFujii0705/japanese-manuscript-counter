## Japanese Manuscript Counter
日本語の文字数と400字詰め原稿用紙換算をリアルタイム表示するObsidianプラグインです。
### 機能
- リアルタイム文字数カウント: ステータスバーに文字数と原稿用紙枚数を表示
- 選択範囲のカウント: テキストを選択すると、選択範囲の文字数と枚数を表示
- 詳細情報の表示: コマンドパレットから詳細なカウント情報を確認可能
- 日本語に最適化: 改行、段落、禁則処理を考慮した正確なカウント

### インストール方法
- コミュニティプラグインから（推奨）
1. Obsidianの設定を開く
2. 「コミュニティプラグイン」→「閲覧」
3. "Japanese Manuscript Counter" を検索
4. 「インストール」→「有効化」

### 手動インストール
1. 最新リリースから main.js と manifest.json をダウンロード
2. Vaultの .obsidian/plugins/japanese-manuscript-counter/ フォルダを作成
3. ダウンロードしたファイルをフォルダに配置
4. Obsidianを再起動し、プラグインを有効化

### 使い方
**ステータスバー表示**
- プラグインを有効化すると、ステータスバー（画面下部）に以下の情報が表示されます：
- 📝 1,234字 (3.09枚)
**選択範囲のカウント**
- テキストを選択すると、選択範囲の文字数と枚数がステータスバーに表示されます。
**詳細情報の確認**
1. コマンドパレット（Ctrl/Cmd + P）を開く
2. "Japanese Manuscript Counter: Show Details" を検索
3. 詳細なカウント情報がモーダルで表示されます

### カウント方法
400字詰め原稿用紙のルールに基づいてカウントします：

全角文字: 1文字
半角文字: 0.5文字
改行・段落: 原稿用紙のルールに準拠
禁則処理を考慮
